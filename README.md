# 大二暑假練習 C++

本儲存庫記錄了大二暑假自主用 AI 學習 C++ 與物件導向程式設計（OOP）的實作成果。
題目都是AI給的，我自己再慢慢摸索，不動就一直問。


### 1. 自動販賣機 (`vending_machine.cpp`) 
* 2026/6/22 完成
* 深入理解 **class** 內部的 **private** 與 **public** 存取權限控制。
* 掌握 **建構子(Constructor)** 的運作機制與物件初始化流程。
* 學習實踐物件導向的特性：**封裝 (Encapsulation)**。
* **程式運作說明** :
*   **輸入** : 首行輸入一個整數，代表預計建立的販賣機物件數量。接下來每行包含一個字串（飲料名稱）與一個整數（販賣機裡面一開始有多少錢），以空格分隔。
*   **建構子資料驗證機制** : 若金額為負數，並將該物件的金額**強制歸零**。
*   **輸出** : 最終結果，會判斷要用 dollar 還是 dollars。

### 2. 簡易學期成績計算系統 (`GradeCalculator.cpp`) 
* 2026/6/23 完成
* 進階運用 **建構子初始化列表 (Constructor Initialization List)**，寫出更高效、符合現代規範的 C++ 程式碼。
* 優化 **浮點數(Double)** 的型態對齊。
* **程式運作說明** :
*   **輸入** : 輸入學號，期中考成績和期末考成績，以空格隔開
*   **輸出** : 最終成績 = 期中考 X 40% + 期末考 X 60%，若最終成績>=60，則pass，反之為fail。

### 3. 城市追蹤與座標重置 (`city_tracker.cpp`)
* 2026/6/24 完成
* **曼哈頓距離** : 二維空間內，計算兩點之間的距離時， **只走直線不走斜線的絕對值距離**。
* 運用了 **結構體** 與資料 **封裝**。
* 使用了 **氣泡排序法 (Bubble Sort)** 來比較大小。
* **傳址呼叫**與**指標操作**與struct的運用，理解了 `*` 與 `&` 與 `->` 的運作。
* **程式運作說明** :
*   **輸入** : 第一行的整數為基地數量 (time)，接下來每行包含一個字串（基地名稱）與兩個整數（X, Y 座標）。當time=0，停止運作。
*   **輸出** : 依據與第一個基準點的曼哈頓距離，由近到遠依序輸出基地名稱與距離數字。

### 4. 雷達掃描與目標鎖定系統 (`radar_lock.cpp`)
* 2026/6/26 完成 (跟 *城市追蹤與座標重置* 類似，感覺不太熟)
*  **切比雪夫距離** = max(|x-cx|,|y-cy|)。
*  手搓了一個我覺得比較複雜的 **Bubble Sort** 。
* **程式運作說明** :
*   **輸入** : 第一行輸入目標總數，接下來每行分別為目標代號，座標X，座標Y，危險程度，最後一行單獨輸入全新的核心座標(cx,cy)，且不再回圈 內。
*   **輸出** : 優先判斷切比雪夫距離，若相同再判斷危險程度，若也相同則依照原輸入順序輸出。

### 5. RPG 角色屬性系統 (`RPG_Character_System.CPP`)
* 2026/6/27 完成
* 初步了解了 `enum` ，並與 `union` 與 `enum` 的概念並實際運用。
* 用了 `switch-case` 的判斷。
* **程式運作說明** :
*   **輸入** : 第一行為角色數量，接下來每一行輸入角色格式為 [職業代碼] [屬性值]。
*   戰士(W:Warrior)-->對應輸入整數憤怒值 (Anger)
*   法師(M:Mage)   -->對應輸入浮點數魔法力 (Magic)
*   盜賊(T:Thief)  -->對應輸入字元暗殺標記 (Sign)
*   **輸出** : 輸出兩個數值，中間以空格隔開。
*   `1.` 所有法師的平均魔法力（固定顯示至小數點後第 2 位；若無人則安全輸出 0.00）。
*   `2.` 帶有 'Y' 暗殺標記的盜賊總人數。

### 6. 攝氏華氏溫度計算 (`Changing Temperature.CPP`)
* 2026/6/29 完成 (維持手感)
* 公式: `C=(F-32)*(5/9)` ， `F=C*(9/5)+32` 。
* 用了書上面的題目與簡易的struct來寫。
* **程式運作說明** :
*   **輸入** : 會先輸出選項:按1是選擇華氏溫度，下一行則輸入華氏溫度，按2則是選擇攝氏溫度，下一行則輸入攝氏溫度，都不是則會輸出WRONG。
*   **輸出** : 運用 if-else 以及 struct 來做計算，若選擇1，則在同一行輸出 : 華氏溫度(溫度) = 攝氏溫度(溫度)，反之，則輸出 : 攝氏溫度(溫度) = 華氏溫度(溫度)。若不是選擇 1 或是 2 則輸出 WRONG。
